# Diwali_Sales_Analysis

# Project Overview

This project analyzes Diwali sales data to uncover insights about customer behavior, purchasing patterns, high-performing categories, and factors that influence sales.

# Objectives

- Clean raw Diwali sales data 
- Perform exploratory data analysis (EDA)
- Identify which Gender shops the most and they are from which age group and from which occupation
- Discover which customer groups contribute most to sales

# Technologies Used

1. Python
2. Pandas
3. NumPy 
4. Matplotlib & Seaborn
5. Jupyter Notebook

# Project Structure

Diwali_Sales_Analysis/
│── Diwali_Sales_Analysis.ipynb     
│── README.md                        
│── Diwali Sale Data.csv                          

# Key Steps Performed

- Handled missing values
- Removed unnecessary columns
- Corrected data types
- Gender-wise purchase behavior
- Age group distribution
- Top occupations purchasing during Diwali
- Bar charts
- Count plots
- Bar graphs
- Category-wise comparisons

# Conclusion

Married women age group of 26-35 yrs from UP,  Maharastra and Karnataka working in IT, Healthcare and Aviation are more likely to buy products from Food, Clothing and Electronics category.
